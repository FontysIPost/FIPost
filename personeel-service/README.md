# personeel-service

.NET Core 3.1 API service for Fontys Internal Packages.

## Getting Started with Docker
```zsh
docker-compose up --build
```

<b>Note</b><br/>
Make sure that you are in the directory of `docker-compose.yml` when running these commands.
